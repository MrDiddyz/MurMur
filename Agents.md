# MurMur Agents

MurMur is built around a system of collaborative AI agents.

Each agent has a specialized role.

---

## Core Agents

### God Agent
The orchestrator of the MurMur system.

Responsibilities:
- dispatch tasks
- coordinate agents
- manage workflow

---

### Research Agent
Finds information and gathers knowledge.

Responsibilities:
- research topics
- gather sources
- summarize findings

---

### Analyst Agent
Analyzes information and identifies patterns.

Responsibilities:
- analyze data
- detect correlations
- extract insights

---

### Memory Agent
Stores knowledge and results.

Responsibilities:
- store outputs
- retrieve past context
- maintain knowledge base

---

### Builder Agent
Responsible for implementation tasks.

Responsibilities:
- write code
- generate structures
- assist development

---

## Council Agents

MurMur can also use a council system where multiple agents debate ideas.

Example council roles:

- Scientist Agent
- Strategist Agent
- Philosopher Agent
- Engineer Agent
- Creative Agent

The council votes on the best solution.

---

## Future Agents

Planned agents include:

- Narrative Agent (media analysis)
- Creator Agent (media generation)
- Orchestrator Agent (advanced task flow)

---

MurMur explores collective intelligence through multiple collaborating agents.
